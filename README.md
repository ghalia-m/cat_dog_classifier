# Cat vs Dog AI Image Classifier

A modern web application for classifying cat and dog images using two different neural network models.

## Features

**Dual Model Comparison**: Simple CNN vs MobileNetV2 (Transfer Learning)
**Modern UI**: Warm yellow palette with animations
**Real-time Processing**: See neural networks in action
**Visual Analytics**: Interactive charts and confidence bars
**Responsive Design**: Works on all devices
